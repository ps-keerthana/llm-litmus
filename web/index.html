How The Platform Evaluates AI Answers

Zero technical knowledge required! Follow the 4-step RAG evaluation flow below.

1

Knowledge Base Ingestion

Ingests 8 official tax reference guides (Section 80C, 80D, 24b, HRA, TDS, Capital Gains, NPS) split into 49 text chunks.

2

Vector Search (Top-5)

Converts user queries into dense vector embeddings using all-MiniLM-L6-v2 and retrieves top 5 relevant document chunks.

3

LLM Deductive Synthesis

The LLM model (Llama 3.3 70B / Llama 3.2 1B) applies tax rules to generate precise advice.

4

Automated Quality Gate

Benchmarks model answer against ground truth across semantic similarity, context recall, and LLM-as-a-Judge correctness.
