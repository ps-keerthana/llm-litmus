Architecture

The 4-Step Evaluation Loop

From raw tax documents to a graded AI answer — fully automated, zero manual review required.

01
📄
Document Ingestion
8 official Indian tax guides — Sec 80C, 80D, Section 24b, HRA, TDS, NPS, Capital Gains — split into 49 semantic chunks and stored in ChromaDB.
02
🔍
Vector Retrieval (k=5)
Each query is encoded via all-MiniLM-L6-v2 embeddings. The 5 most semantically relevant chunks are retrieved.
03
🤖
LLM Answer Synthesis
Llama 3.3 70B via Groq (remote) or Llama 3.2 1B via Ollama (local) generates a precise answer strictly grounded in retrieved context.
04
⚖️
Automated Quality Gate
Answers are scored on semantic similarity, faithfulness, and hallucination. CI blocks merges below 70% pass rate automatically.
